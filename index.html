Hi David,by default, an image is rendered inline, like a letter so it sits on the same line that a, b, c and d sit on. There is space below that line fot descenders you find on letters like g, j, p and q.

You can:

• adjust the vertical-align of the imageto position it elsewhere (e.g. the middle ) or
• change the display so it isn't inline.
